ชุดที่ต้องสอบซ่อม

กรุณาเลือกชุดข้อสอบซ่อมทางด้านซ้าย

คู่มือการเชื่อมต่อฐานข้อมูล Google Sheets

  1. สร้าง Google Sheet ใหม่:

    เปิด Google Sheets ขึ้นมา 1 แผ่น ว่างเปล่า

  2. เปิด Apps Script:

    ไปที่เมนูด้านบน เลือก ส่วนขยาย > Apps Script

  3. วางโค้ด API:

    เปิดไฟล์ google_apps_script.js ในโปรเจกต์ คัดลอกโค้ดทั้งหมด ไปวางทับลงในหน้าต่างของ Apps Script

  4. บันทึกและตั้งค่าตารางเริ่มต้น:

    คลิกปุ่มบันทึก (แผ่นดิสก์) จากนั้นเลือกฟังก์ชัน setup ในช่องเลือกฟังก์ชัน และกดปุ่ม "เรียกใช้งาน (Run)" ด้านบน เพื่อสร้างชีทที่จำเป็นอัตโนมัติ

  5. Deploy เป็น Web App:

    คลิกปุ่ม ทำให้ใช้งานได้ (Deploy) > การปรับใช้ใหม่ (New deployment) เลือกประเภทเป็น "เว็บแอป (Web App)"

    • เรียกใช้งานในฐานะ: ฉัน (อีเมลของคุณครู)
    • ผู้มีสิทธิ์เข้าถึง: ทุกคน (Anyone)

    คลิก ทำให้ใช้งานได้ (Deploy) และยืนยันสิทธิ์อนุญาตความปลอดภัยของ Google ให้เสร็จสิ้น

  6. นำ URL มาเชื่อมต่อ:

    คัดลอก **URL ของเว็บแอป (Web App URL)** ที่ระบุลงท้ายด้วย /exec แล้วเปิดไฟล์ config.js นำ URL นั้นมาใส่ในตัวแปร API_URL และกดบันทึก
